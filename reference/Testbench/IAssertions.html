IAssertions

Namespace
Testbench
INTERFACE IAssertions

Collection of utility methods that support asserting conditions in unittests. The declarations are taken from TcUnit and the usual implementation of this interface that is used when testing is a wrapper around FB_TestSuite.

For details, see IUnitTest.

EqualsArray2dLreal 

METHOD PUBLIC EqualsArray2dLreal (
 [input] delta : LREAL,
 [input] message : STRING(255),
 [inout] expected : ARRAY[*,*] OF LREAL,
 [inout] actual : ARRAY[*,*] OF LREAL)

Asserts that two LREAL 2D-arrays are equal to within a positive delta. If they are not, an assertion error is created.

Inputs

delta LREAL

The maximum delta between the value of expected and actual for which both numbers are still considered equal, proportional to the expected value in that array cell

message STRING(255)

The identifying message for the assertion error

InOuts

expected ARRAY[*,*] OF LREAL

LREAL 2d array with expected values

actual ARRAY[*,*] OF LREAL

LREAL 2d array with actual values
